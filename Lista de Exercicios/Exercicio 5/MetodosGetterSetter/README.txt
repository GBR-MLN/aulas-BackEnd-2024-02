Na classe Pessoa é criado um atributo privado "nome" que só pode ser acessado pela classe, então é criado um constructor, 
em seguida são criados os métodos getter e setter, esses métodos são criados para manipulação do atributo por outras classes,
de volta na classe principal, um objeto Pessoa é criado com o nome "Ana" Atribuido a ele, é então chamado o método "dizerOla()"
e é retornado uma String com o nome "Ana", então é utilizado o método "setNome" para alterar o nome para "Beatriz", o método "dizerOla()"
é chamado novamente e a String retorna com o nome "Beatriz".
