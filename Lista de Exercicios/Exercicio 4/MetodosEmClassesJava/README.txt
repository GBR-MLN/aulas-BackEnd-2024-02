Na classe pessoa é criado um atributo (nome), então é criado um constructor para o atributo, em seguida é criado um método que
printa uma String usando o nome atribuido ao objeto Pessoa, de volto na classe principal é criado um objeto Pessoa e então é dado
um nome ao objeto, em seguida é chamado o método criado na classe Pessoa que mostra a String com o nome atribuido ao objeto.
